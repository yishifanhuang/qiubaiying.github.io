>Markdownn编辑器学习

# 工具介绍
Markdown是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，可以简单的使普通文本内容具有一定的格式化。

| 名称     | Markdown   |  宗旨  | 易读易写
| -------- | :-----:  | :----:  |:----: |
| 应用   |文档快速排版  |   特点   | 轻量、简单、通用
|   格式  |    LaTeX，Docbook  |   开发者   |    John Gruber   |
|      |       |  文件后缀名   |.md;.markdown;.mdown       |
